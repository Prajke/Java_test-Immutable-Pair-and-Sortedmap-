A simple test to make sure that you know the type system in Java.

* Implement the classes Pair and SortedMap however you like.
* Keep it short and simple.
* Refactor and rewrite as you wish.
* Defend your design choices in javadoc.

The expected functionality can be deduced from Javadoc and unit tests.

If you are familiar with Maven you can use that to get started.
If you don't have Maven installed, the project contains two source folders:
src/main/java and src/test/java. The dependencies needed are simply Java8 and
JUnit 4.11.
